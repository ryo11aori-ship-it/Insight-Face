# 顔一致度分析システム

同一人物の資料顔写真A群(1〜複数枚)を登録し、検体顔写真Bが同一人物である確率を算出するローカル動作のシステムです。

## 技術構成(オープンソース最高水準)

- **顔検出**: RetinaFace(様々な角度・サイズの顔に頑健)
- **顔認証**: ArcFace ResNet100(InsightFace buffalo_l)— LFWベンチマーク精度 99.8%超で、オープンソースで利用できる最高峰クラスのモデル
- **複数枚統合**: A群の平均埋め込み(センタロイド)+ 上位個別類似度を統合し、1枚照合より高精度
- **確率較正**: ArcFaceのコサイン類似度分布に基づくロジスティック較正で確率(%)を出力
- **完全ローカル**: 照合処理はすべて自分のPC内で完結(初回のみモデル約300MBを自動ダウンロード)

## 対応形式

JPG / PNG / WebP / BMP / TIFF / GIF / HEIC / HEIF / AVIF / RAW(rawpy導入時)など。
サイズは自由(巨大画像は自動縮小、小さい顔は自動拡大再検出)。EXIF回転も自動補正。

## セットアップ

Python 3.9〜3.11 推奨。

```bash
pip install -r requirements.txt
```

- NVIDIA GPUがある場合: `pip uninstall onnxruntime` → `pip install onnxruntime-gpu` で高速化
- Windowsで insightface のビルドに失敗する場合: Visual Studio Build Tools(C++)を導入するか、
  `pip install insightface --prefer-binary` を試してください

## 使い方

### GUIモード(推奨)

```bash
python app.py
```

ブラウザが自動で開きます。
1. 「資料写真A群」に同一人物の写真を複数枚ドラッグ&ドロップ →「A群を登録」
2. 「検体写真B」に1枚ドラッグ&ドロップ →「分析実行」
3. 同一人物確率(%)、判定、写真ごとの類似度が表示されます

### CLIモード

```bash
python app.py --ref 資料フォルダ --probe B.jpg
python app.py --ref a1.jpg a2.heic a3.png --probe B.webp
```

集合写真など小さい顔が多い場合は `--det-size 1024` を追加してください。

## 精度を最大化するコツ

- A群は **角度・照明・表情の異なる3〜10枚** を用意すると確率推定が安定します
- 顔が概ね **80px四方以上** 写っている写真を使う
- A群に別人が混入するとシステムが警告を出します(自動検出)

## 出力の見方

| 確率 | 目安 |
|---|---|
| 98%以上 | 同一人物である可能性が極めて高い |
| 90〜98% | 非常に高い |
| 70〜90% | 高い |
| 40〜70% | 判定困難(資料写真の追加を推奨) |
| 40%未満 | 別人の可能性が高い |

## 注意事項

- 出力される確率は統計的推定値であり、法的証明・本人確認の唯一の根拠として使用しないでください
- 双子・近親者・大幅な経年変化・強い加工(美肌アプリ等)では精度が低下します
- 他者の顔写真を分析する際は、本人の同意・プライバシー・各国の個人情報保護法(顔情報は生体情報として厳格に規制される場合があります)を遵守してください
